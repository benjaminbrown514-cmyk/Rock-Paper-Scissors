<# Rock-Paper-Scissors game>
## Description 
Homework 4: Rock, Paper, Scissors
A browser-based implementation of the classic "Rock, Paper, Scissors" game built entirely with JavaScript. This application uses standard browser methods to interact with the user and features an automated opponent with randomized logic.

üïπÔ∏è How to Play
Start the Game: Open the application in your browser.

Make Your Move: When prompted, enter R (Rock), P (Paper), or S (Scissors).

The Opponent: The computer will randomly select its own move.

View Results: An alert will display whether you won, lost, or tied the round.

Track Stats: Your total wins, losses, and ties are updated and displayed after every round.

Play Again: Choose to start a new round or exit the game via the confirmation dialog.

üìã User Stories
Play Against AI: As a user, I want to play Rock, Paper, Scissors against an automated opponent.

Input Choice: As a user, I can enter R, P, or S to signify my choice.

AI Choice: As a user, I expect the computer to choose R, P, or S in return.

Persistence: As a user, I want the option to play again whether I win or lose.

Statistics: As a user, I want to see my total wins, ties, and losses after each round.

üõ†Ô∏è Technical Specifications
This project is built using vanilla JavaScript and relies on the following browser-native methods for the user interface:

prompt(): Used to capture the player's move (R, P, or S).

alert(): Used to display the outcome of the round and the running score.

confirm(): Used to ask the player if they wish to play another round.

Logic Flow

The game determines the winner based on the standard rules:

Rock beats Scissors

Scissors beats Paper

Paper beats Rock

The computer's choice is generated using Math.random() to ensure a fair and unpredictable experience.
